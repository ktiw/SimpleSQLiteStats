# AdvancedStats
## Описание:
Система статистики игроков (Убийства/Смерти/KDR) с хранением данных в Базе Данных.
Используется асинхронное подключение (Async), чтобы сервер не лагал при запросах к базе. Поддерживает топ игроков.
## Команды:
/stats — Посмотреть свою статистику.
/stats top — Топ-10 игроков сервера.
